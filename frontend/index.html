操作手册

地图基础操作

缩放与移动

  • 滚轮缩放 — 滚动鼠标滚轮放大/缩小地图
  • 拖拽平移 — 按住鼠标左键拖拽移动地图
  • 右键菜单 — 右键点击地图弹出操作菜单

坐标信息

  • 移动鼠标时底部状态栏显示当前经纬度
  • 视图 → 坐标信息 可开启/关闭左下角常驻坐标显示
  • 底部坐标文字点击可复制到剪贴板

绘制工具

几何图形绘制

  • 点击右侧快捷栏的绘图按钮（多边形/矩形/圆/线/标记）进入绘制模式
  • 在地图上依次点击生成图形，完成后自动添加到图层列表
  • 绘制按钮右下角 绿色 表示当前选中
  • 按 Esc 取消绘制
  • 也可通过顶部菜单 绘制 操作

选择要素

  • 点击 选择要素 按钮激活工具
  • 光标变为十字准星，点击已绘制的图形弹出属性卡片
  • 再次点击按钮或按 Esc 退出

删除要素

  • 点击 删除要素 按钮进入删除模式
  • 点击图形将其从地图和图层面板中移除
  • 再次点击按钮或按 Esc 退出删除模式

图层管理

图层面板

  • 点击快捷栏底部的 图层图标 打开/关闭图层面板
  • 面板位于地图右上角，列出所有已加载的图层
  • 每行显示图层名称、类型、可见性开关（ 点击切换）
  • 图层行悬停显示操作按钮：属性表、改色、缩放至、删除

上传文件

  • 支持格式：GeoJSON、SHP（zip 包）、GPKG、KML/KMZ、GPX、DXF、CSV
  • 可通过图层面板的「上传」按钮或输入框旁的附件按钮上传
  • 上传后自动注册为图层，AI 可直接识别和使用

图层操作

  • 在聊天中告诉 AI 操作图层：「删除图层 xxx」「隐藏 xxx」「把 xxx 改成红色」「缩放到 xxx」
  • AI 调用 layer_control 工具执行：移除、显隐切换、改色、重命名、缩放至

导出图层

  • 在聊天中输入「导出 xxx 为 Shapefile」或「导出为 GeoJSON」
  • 也通过斜杠命令 /export 操作
  • SHP 导出自动打包为 zip（含 .shp .shx .dbf .prj .cpg）

字段计算

  • 在聊天中输入「给图层加一个面积字段」「计算长度」「创建新字段」
  • 支持 Python 表达式，直接引用已有字段名
  • 结果自动更新到地图和图层面板

统计图表

  • 选中图层后说「统计各类型的数量」「做柱状图」
  • 支持柱状图、饼图、直方图、散点图、折线图
  • 图表以 ECharts 交互式 HTML 显示在聊天中，可导出 SVG

属性表

  • 在图层面板选中图层后，点击 属性表 查看要素属性
  • 支持编辑、筛选、导出功能
  • 每行末的 ◎ 按钮可定位到地图对应要素
  • 也通过顶部菜单 工具 → 属性表 打开

顶部菜单栏

地图顶部的菜单栏提供系统所有功能的入口。

文件

  • 导入图层 — 上传文件到地图（GeoJSON / SHP / GPKG / KML 等）
  • 导出 — 导出当前选中的图层为 GeoJSON
  • 保存工程 — 保存当前会话为工程文件（含图层 + 对话 + 设置）
  • 加载工程 — 加载已保存的工程，恢复工作现场

绘制

  • 列出所有绘图工具：多边形、矩形、圆、线、标记
  • 第一行 添加至快捷栏 开关控制右侧工具栏是否显示绘图按钮

视图

  • 图层面板 — 切换面板显隐
  • 坐标信息 — 持久开关，控制左下角坐标显示
  • 卫星底图 — 切换卫星影像/普通地图底图

工具

  • 选择要素 — 点击地图要素查看属性
  • 第一行 添加至快捷栏 开关控制选择要素/删除按钮显隐

AI 对话

发送消息

  • 在左侧聊天面板输入问题，按 Enter 或点击发送按钮
  • 支持斜杠命令（输入 / 弹出命令面板，点击或回车选择）
  • AI 可生成地图数据、分析地理信息、搜索网络等
  • 右键地图可「发送此位置给 AI」，自动携带坐标

模型选择

  • 聊天面板顶部下拉框选择 AI 模型
  • 支持 DeepSeek V4 Flash+、GLM-4.7-Flash+、Agnes 2.0 Flash+
  • 齿轮按钮可配置各模型的 API Key

斜杠命令一览

在输入框输入 / 弹出命令面板，支持以下快捷操作：

命令 说明
help打开此操作手册
buffer为指定图层创建缓冲区
intersection两个图层的空间相交分析
union合并两个图层的几何
difference一个图层减去另一个图层
centroid提取图层的质心/中心点
simplify简化图层几何，减少顶点数
area计算图层面积（自动选投影）
length计算线图层长度
make_valid修复无效几何图形
aoi提取地点建筑轮廓（百度 AOI）
boundary获取省/市/区行政边界
heatmap从点图层生成热力图
plot生成数据统计图表
chart图层属性统计图表（柱状/饼图/散点）
export导出图层为 Shapefile 或 GeoJSON
amap搜索 POI / 地址转坐标 / 查天气（高德）
network路网分析（最短路径 / 服务区 / 最近设施）

常用 AI 功能

  • POI 搜索 — 「搜索广州的医院」「附近有哪些便利店」
  • 地理编码 — 「广州塔的坐标」「北京西站的位置」
  • 下载路网 — 「下载广州市的路网」「下载北京的路网数据」
  • 路径规划 — 「从广州塔到广州南站最短路径」
  • 获取边界 — 「北京市的行政边界」「天河区的边界」
  • 提取 AOI — 「提取广州塔的建筑轮廓」「AOI 珠江新城」
  • 字段计算 — 「给图层加一个面积字段」「计算长度」
  • 图表分析 — 「统计各类型 POI 的分布」「做成饼图」
  • 面积测量 — 「测量这个图层的面积是多少平方公里」
  • 搜索网络 — 「查一下广州最新的人口数据」

历史与工程

  • 点击聊天面板右上角时钟按钮查看历史会话
  • 支持新建会话、全部清除
  • 工程功能可保存/恢复完整工作状态（含地图图层和对话）

空间分析工具

右侧快捷栏底部提供了专用的空间分析工具按钮。

网络分析（路径/服务区/最近设施）

  • 点击右侧工具栏的 网络分析 按钮（折线图标）打开分析面板
  • 支持三种分析类型：
    • 最短路径 — 设起点和终点，计算沿路网的最短驾驶路径
    • 服务区 — 设设施点，计算沿路网可到达的范围（如 1km / 3km / 5km）
    • 最近设施 — 设事件点和多个设施点，找出最近的 N 个设施
  • 起终点/设施点可在输入框输坐标，也可点击「在地图上选点」后直接点地图
  • 分析结果自动生成图层加载到地图和图层面板
  • 需要先有路网图层（上传 OSM 路网数据或通过 AI 下载）

AOI 建筑轮廓提取

  • 在 AI 聊天中输入「提取 XX 的建筑轮廓」或「AOI XX」
  • AI 会搜索候选地点并显示在聊天框，点击选择即可提取
  • 数据来源为百度地图，坐标自动转为 WGS-84
  • 提取失败时 AI 会如实告知，不会自行估算边界

行政边界获取

  • 在 AI 聊天中输入「获取 XX 的行政边界」
  • 支持中国省/市/区三级行政区划（数据来源：阿里云 DataV）
  • 坐标自动转为 WGS-84，结果自动加载到地图
  • 国外地区边界请用 AI 的 execute_python + osmnx 获取

热力图

  • 先有点图层（含权重字段），然后在聊天中说「生成热力图」
  • 可指定图层名、权重字段、像素半径、渐变色
  • AI 会自动处理并返回热力图层
